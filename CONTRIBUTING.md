# Contributing to RAG-Pinecone

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/<your-username>/RAG-Pinecone.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Install dev dependencies: `pip install -r requirements-dev.txt`
5. Make your changes and add tests
6. Run tests: `make test`
7. Push and open a Pull Request

## Code Style

- Follow PEP 8 (enforced via `pre-commit`)
- Run `make lint` before committing
- All new features need tests under `tests/`

## Reporting Bugs

Open a GitHub Issue with:
- Steps to reproduce
- Expected vs actual behavior
- Python version and OS
